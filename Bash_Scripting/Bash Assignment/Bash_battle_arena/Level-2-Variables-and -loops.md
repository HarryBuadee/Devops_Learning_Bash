Level 2: Variables and Loops

Mission: Create a script that outputs the numbers 1 to 10, one number per line.

Making text document named count.sh


what I inputted in the text document:
```
#!/bin/bash

for i in {1..10}
do
  echo $i
done
````


To make it an executable:

```
chmod +x count.sh

````

Run it:

````
./count.sh

````
